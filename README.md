Classification and Regression Tree (CART)
====
Jianbo Ye

This is a sample implementation of CART in Scala, which includes
 - Splitting Rules
 - Class Assignment
 - Pruning Trees

To run
```bash
$ sbt test:run
$ dot -Tpng graphTrain.dot -o graphTrain.png; open graphTrain.png
```
